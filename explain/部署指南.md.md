# 部署指南.md — 从零部署教程

## 文件定位
面向零环境的新手，分步骤指导安装 Python、MySQL、MinGW，最终启动项目。

## 内容结构
1. 第一步：安装 Python 3（附 python.org 下载链接和 PATH 配置说明）
2. 第二步：安装 MySQL 8.0（Root 密码设为 zjy200646）
3. 第三步：安装 C++ 编译器 MinGW-w64（可选，如 judge.exe 已编译可直接用）
4. 第四步：4 条命令跑起来
   - pip install -r requirements.txt
   - mysql -u root -pzjy200646 < db/schema.sql
   - g++ -o judge.exe judge.cpp -static -O2 -lpsapi (可选)
   - python app.py
5. 第五步：浏览器访问 http://127.0.0.1:5000
   - 默认账号：admin/admin123 (管理员) / student1/123456 (学生)
6. 常见问题 6 条：pip 报错、mysql 连接失败、g++ 找不到、端口占用、500 错误、pending 状态
